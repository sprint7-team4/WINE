## 개요
- 프로젝트 이름: Wine 🍷
- 프로젝트 지속기간: 2024.07.29 ~ 2024.08.14
- 멤버: 홍준기, 임창기, 서미영, 김당찬

## 서비스 소개
- 사용자가 경험해 봤던 와인을 등록.
- 다른 사용자들은 이 와인을 구경하고 자유롭게 후기를 남김
- 서로의 다양한 느낌을 공유하고 소통하는 공간을 제공.
